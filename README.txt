ATM SIMULATOR PROJECT

Project Structure:
- index.html: Main homepage
- login.html: User login page
- balance.html: Balance inquiry page
- withdraw.html: Money withdrawal page
- deposit.html: Money deposit page
- loan.html: Loan debt viewing page
- minimum-payment.html: Minimum payment calculation page
- styles.css: Styling for all pages
- script.js: JavaScript functionality for all pages

How to Run:
1. Open index.html in a web browser
2. Navigate through the pages using the menu
3. Login with:
   Account Number: 123456789
   PIN: 1234
4. Perform various ATM operations

Features:
- User authentication
- Balance inquiry
- Money withdrawal
- Money deposit
- Loan debt viewing
- Minimum payment calculation

Frontend Technologies Used:
- HTML5
- CSS3
- JavaScript (ES6)

Backend:
- PHP (to be implemented in future phases)

Note: This is the frontend implementation only. Data is stored in-memory and will reset on page refresh.
